# RTK - Token Optimization

Use RTK to reduce token burn in coding-agent sessions when installed.

## Preferred Commands

```powershell
rtk git status
rtk git diff
rtk read README.md
rtk grep "pattern" .
rtk find "*.md" .
rtk test npm test
rtk pytest
rtk lint
rtk tsc
rtk deps
rtk gain --history
```

RTK does not make sensitive output safe. Never print secrets, production credentials, raw PII, customer lists, claim notes, tokens, or connection strings.
