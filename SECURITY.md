# Security Policy

## Scope

VietActuaryKit is a public knowledge-base template. It should not contain secrets, credentials, API keys, customer data, claim data, policy data, internal company workflows, proprietary pricing assumptions, or production configuration.

## Reporting a Problem

If you find sensitive information or a security issue in this template, open a private security advisory if available, or contact the repository maintainer directly.

## Safe Use

Before publishing your own fork, search for:

- Local user paths or workspace-specific folders.
- API keys, tokens, connection strings, private keys, passwords.
- Customer, policy, claim, employee, agent, broker, partner, hospital, garage, or beneficiary identifiers.
- Company-specific pricing, reserving, reinsurance, commission, profitability, or underwriting logic.

Do not use this repository as legal, compliance, security, actuarial, or insurance advice.
